简介

Midway 是阿里巴巴 - 淘宝前端架构团队，基于渐进式理念研发的 Node.js 框架，通过自研的依赖注入容器，搭配各种上层模块，组合出适用于不同场景的解决方案。

Midway 基于 TypeScript 开发，结合了面向对象（OOP + Class + IoC）与函数式（FP + Function + Hooks）两种编程范式，并在此之上支持了 Web / 全栈 / 微服务 / RPC / Socket / Serverless 等多种场景，致力于为用户提供简单、易用、可靠的 Node.js 服务端研发体验。

项目结构

TypeORM

TypeORM 是一个支持多种数据库的 TypeScript（也可以使用 JavaScript）对象关系映射（ORM）库。它提供了一种方便的方式来进行数据库操作，包括创建表、插入、更新和查询数据等。TypeORM 可以与多种数据库系统集成，例如 MySQL、PostgreSQL、SQLite、Microsoft SQL Server 等。

使用 TypeORM，可以通过定义实体（Entities）来映射数据库表，定义实体属性（Columns）来映射表的列，还可以定义关系（Relationships）来建立表之间的关联。TypeORM 还提供了丰富的查询，可以进行复杂的查询操作，并支持事务、连接池等高级功能。

下面列一下我们用到的几个类

  • @EntityModel 用来定义一个实体类；
  • @Column 用来描述类的一个熟悉，对应数据库就是一个数据列；
  • @PrimaryColumn 用来定义一个主键，每个实体类必须要要主键；
  • @PrimaryGeneratedColumn 用来定义一个自增主键；
  • @CreateDateColumn 定义创建时，自动设置日期；
  • @UpdateDateColumn 定义更新时，自动设置日期；
